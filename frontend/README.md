# SkillVerse Frontend

## Commands

```bash
npm install
npm run dev
```

## Deployment on Vercel

1. Import the `frontend/` folder
2. Build command: `npm run build`
3. Output directory: `dist`
4. Set `VITE_API_URL` to your deployed backend URL

## Design Notes

- Government education visual language with premium gradients and glass cards
- Responsive experience across mobile, tablet, and desktop
- Light and dark mode
- English, Hindi, and Kannada language toggles
